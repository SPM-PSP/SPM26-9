# 测试报告：test_logger.py（日志模块）

## 一、测试对象信息
- **模块路径**：`core/logger.py`
- **核心类/函数**：`Logger`、`init`、`get`、`debug`、`info`、`warning`、`error`
- **模块功能概述**：
  日志模块负责插件运行过程的**日志记录、级别过滤、文件持久化、日志查询**，支持多级别日志输出（DEBUG/INFO/WARNING/ERROR），便于问题排查、运行追溯与异常定位。

## 二、测试目的
1. 验证日志**基础写入、文件生成、内容正确性**；
2. 验证**日志级别过滤**功能生效，低级别日志被正确拦截；
3. 验证**最近日志查询**功能返回结果准确；
4. 确保日志IO操作**稳定可靠、无丢失、无报错**；
5. 保证日志模块**独立运行、无外部依赖干扰**。

## 三、测试环境
- 操作系统：Windows
- 运行环境：Python 虚拟环境
- 测试框架：pytest
- 测试依赖：使用临时目录模拟日志存储路径
- 外部依赖：无
