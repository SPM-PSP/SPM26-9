# 测试报告：test_layer_manager.py（图层管理模块）
**Markdown 标准版｜可直接用于文档、汇报、导出 Word**

---

## 一、测试对象信息
- **模块路径**：`core/layer_manager.py`
- **核心功能**：图层管理工具集
- **模块说明**：
  提供 GIMP 图层操作的统一封装，包括**图层复制、预览层创建、图层查找、可见性切换、新建图层、图层组管理**等，是插件实现非破坏性编辑、预览效果的核心支撑模块。

---

## 二、测试目的
1. 验证图层管理各项基础功能**执行正确**；
2. 验证图层查找、名称匹配、状态切换逻辑**稳定可靠**；
3. 验证模块在**无真实 GIMP 环境**下可通过 Mock 正常测试；
4. 确保图层操作**不崩溃、不报错、符合业务逻辑**；
5. 保证图层创建、填充、插入、命名流程符合 GIMP 规范。

---

## 三、测试环境
- 操作系统：Windows
- 运行环境：Python 虚拟环境
- 测试框架：pytest
- 依赖隔离：使用 `unittest.mock` 模拟 GIMP 图像、图层、Drawable 对象
- 外部依赖：无真实 GIMP 运行时
