## 使用的接口

### GIMP API

| 接口 | 用途 |
|------|------|
| `Gimp.ImageProcedure` | 注册图片处理插件 |
| `Gimp.PlugIn` | 插件基类 |
| `GimpUi.ProcedureDialog` | 对话框基类 |
| `image.get_width() / get_height()` | 获取图片尺寸 |
| `image.get_file()` | 获取图片文件路径 |
| `gimp-file-export` | 导出图片为 PNG（用于发送给 AI） |
| `gimp-image-get-thumbnail` | 获取缩略图（未启用） |

### AI API（兼容 OpenAI 格式）

| 参数 | 当前值 |
|------|--------|
| API 地址 | `https://api.deepseek.com/v1/chat/completions` |
| Model | `deepseek-chat` |
| 认证 | `Authorization: Bearer <api-key>` |
| 图片传输 | Base64 PNG，`image_url` 格式 |

### 请求格式

```json
{
  "model": "deepseek-chat",
  "messages": [
    {"role": "system", "content": "..."},
    {"role": "user", "content": [
      {"type": "text", "text": "..."},
      {"type": "image_url", "image_url": {"url": "data:image/png;base64,..."}}
    ]}
  ],
  "max_tokens": 4096
}
```

## 注意事项

1. **插件路径**：用户插件放 `~/Library/Application Support/GIMP/3.2/plug-ins/ai-mentor/`，不是 .app 包内
2. **文件可执行权限**：`ai-mentor.py` 必须有 `+x` 权限
3. **GIMP 3.2 API 差异**：`Image` 没有 `get_filename()`，改用 `get_file().get_basename()`
4. **界面线程**：AI 请求在后台线程执行，通过 `GLib.idle_add()` 回到主线程更新 UI
5. **配置持久化**：API Key 等设置保存在 `config.json`
6. **缩略图**：当前显示通用图标，后续可以用 `gimp-image-get-thumbnail` 实现真实预览
